# Build Your Launch Kit

---

## The Full Launch Kit

This command builds 4 deliverables:

### Deliverable 1 — Landing Page
`[product]-landing.html`
- Full conversion-focused landing page
- Real copy, bold design, mobile-first
- See: `/solo-maker:landing` for full process

### Deliverable 2 — Email Welcome Sequence
`[product]-email-sequence.md`
5-email sequence:
1. Welcome + delivery
2. Value: one useful insight
3. Story: why you built this
4. Proof: result/case study
5. Offer: clear CTA

Each email: subject line + preview text + full body

### Deliverable 3 — Launch Week Social Kit
`[product]-social-launch.md`

7 days of posts:
- Day 1: The announcement (problem-first)
- Day 2: Behind the story
- Day 3: Show the product
- Day 4: Social proof / testimonial
- Day 5: Answer top objection
- Day 6: Use case / who it's for
- Day 7: Last call + urgency

For each day: Twitter thread + LinkedIn post + Instagram caption

### Deliverable 4 — Brand Quick Guide
`[product]-brand-quick.html`
- Color palette (OKLCH values)
- 2 fonts with CSS
- Voice do/don't (5 examples)
- 5 tagline options

---

## Build Order

1. Gather the 5 questions (solopreneur skill)
2. Write all copy first (copywriting skill)
3. Build landing page HTML
4. Write email sequence
5. Write social launch posts
6. Build brand quick guide

## Deliver

Announce each file as it's completed. At the end:
"Launch kit complete. You have [4] files ready to deploy. Launch checklist: [simple 5-step list to go live]."
