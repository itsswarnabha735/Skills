# Choice Architecture Support Template

**Objective**: Structure a decision to minimize noise and bias.

## 1. Define the Problem (Neutral Framing)

_Describe the choice without assuming the solution._

- Problem: [Context]
- Objective: [What are we optimizing for?]

## 2. Generate Options (Expansion Phase)

_List at least 3 distinct options. "Do nothing" is always an option._

- Option A: [Status Quo]
- Option B: [Change 1]
- Option C: [Change 2]
- Option D: [Wildcard]

## 3. Evaluation Matrix (Selection Phase)

_Rate each option on these independent dimensions (0-10)._

| Dimension | Weight  | Option A | Option B | Option C | Option D |
| --------- | ------- | -------- | -------- | -------- | -------- |
| Impact    | [0.x]   |          |          |          |          |
| Cost      | [0.x]   |          |          |          |          |
| Risk      | [0.x]   |          |          |          |          |
| **Score** | **1.0** |          |          |          |          |

## 4. Pre-Mortem (Hygiene)

_Imagine Option X failed. Why?_

- Selected Option: [Winner]
- Failure Scenario: [Narrative]
- Mitigation: [Prevention step]

## 5. Final Decision

- Choice: [Selected Option]
- Rationale: [Why it won]
- Review Date: [When will we check if it worked?]
